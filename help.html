Help & User Guide

This page explains how to use the Volunteer Management System.

1. Dashboard

The dashboard shows the overall system status such as total volunteers, activities, attendance and certificates.

2. Volunteer List

This module displays all registered volunteers with their ID, name, role and assigned activity.

3. Volunteer Profile

This module shows detailed information of a selected volunteer including personal details, attendance and certificate status.

4. Activity List

This module shows all activities such as Tree Plantation, Blood Donation Camp and Clean India Drive with date and status.

5. Assign Activity

This module is used to assign volunteers to different activities.

6. Attendance

This module is used to mark and view attendance of volunteers as Present or Absent.

7. Certificates

This module shows certificate status of volunteers after completing activities.

8. Reports

This module generates summary reports such as total volunteers, total activities and attendance count.

9. Notifications

This module displays system messages like new activity added, attendance marked and certificate generated.

10. Settings

This module is used to manage system settings such as profile update and notification preferences.
